Welcome to Lockman

A simple folder locker for Windows. Lockman hides a folder and protects it with a password using a small .bat script.

About Lockman

Lockman is a beginner-friendly folder locker built with a Windows batch script (.bat). It lets you create a folder called Locker, then "lock"
